I started my FreeCodeCamp journey...

Browsing through the internet for some free websites where I can learn coding, I stumbled upon FreeCodeCamp, I really liked their friendly user interface so I decided to try it. After the first Responsive Web Design certificate I was hooked. I decided to do one fronted certificate a month.

JavaScript Algorithms and Data Structures certificate was my first encounter with algs and data structures. First dozen lessons were intro to JavaScript so it was preety easy for since I already knew some JavaScript from Udacity courses. Later this algorithms and data structures problems were a bit complex for me back, I really did not unterstand regex and they were based on solving something using regular expressions. On some of them it took me up to 30min to come up with the solution. It was definitely the hardest time in my learning path at that point.

It was a really interesting journey through FreeCodeCamp and I would suggest it to anyone who is just starting to learn frontend developing or coding in general, since the first lessons are focused on HTML and CSS which are not programming languages and are alot easier to learn.

I really liked the Front End Libraries Certification because I felt like I learned something new every lesson. At the time I really wasn't familiar with the frontend frameworks and libraries, so I was fascinated how people created them to make our life as developers easier.

React was my favourite library, although I enjoyed jQuery alot. State management Redux was a bit weird at first, but then everything clicked. I used pure ReactJS for my FCC frontend projects, but I had to supplement with ReactJS documentation and some Youtube videos. At first it was really hard to think in components but I slowly got used to it. I would really like to work with React, now I am learning more and more, about hooks, composition etc.
