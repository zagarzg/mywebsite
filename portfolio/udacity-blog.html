Everything started with Udacity...

I got Udacity recommended by my friend who is an iOS app developer. I started with Intro to HTML and CSS course, since my final project for finishing highschool was a website, it felt preety easy but nevertheless I learned some new stuff. Intro to JavaScript was the next course in which I enrolled. I had some knowledge about programming concepts because I did some C++ in highschool, but I had to get used to the new syntax and JS in general.

This is a perfect site if you are just starting programming. My advice is to check it out. They explain everything in simple terms. And you get to program in their intergrated IDE.

JavaScript and the DOM was a really interesting course where I learned about creating content with JavaScript, working with browser events and performance in general.I jumped to the ES6 course. It was really interesting to see how JavaScript evolved, I learned about the new keywords, arrow functions, Sets, Maps, Proxies, Generators, Iterators and more. Intro to Ajax was the toughest course I enrolled in on Udacity, it was all confusing at the first and a bit overwhelming, but I finally got it.
