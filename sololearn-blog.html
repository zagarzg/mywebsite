Trying out SoloLearn app...

SoloLearn was one app that I wanted to try from the beginning of my learning path, because it reminded me of Duolingo, a foreign language learning app that I used so often.

I really liked the friendly user interface and I was amazed by the user expirience. Since I never studied any of the class based languages I decided I will learn to write some Java. Lessons quiz based so after you learn some theory you do this small quizes or code insertions. I think it is a great way to learn some basic programming concepts.

SoloLearn app is great for beginners who want to learn to code. App will handhold you throughout your journey and you will quickly realize that you are steadily progressing. UX/UI design will motivate you to do the tasks consistently. You will even get some certificates with which you can brag to your parents.

I noticed that people Challenged me, at that time I did not know what it was, basicly you go 1v1 on quizes and you compete with people who will get more right answers, nevertheless I really liked this feature of the app, since I am very competitive.

I wanted to refresh my memory about core web tehnologies HTML, CSS and JavaScript so I enrolled in all three of them on SoloLearn, it was weird to see how I forgot some things about Maps and Sets, but luckily I reminded myself about them through SoloLearn. Nowadays I am doing the React and Redux course and jQuery course.
